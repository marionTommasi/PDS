UTILISATION :

La commande make compile mtail.
Pour utiliser mtail :
       ./mtail n file
avec n le nombre de lignes à afficher du fichier file.

REMARQUES : 

Nous avons préféré implémenter la version utile de tail (question 18), 
mais nous n'avons pas réussi à construire un tampon de taille variable.
